# Placeholder Images Directory

This directory contains placeholder images for the product catalog.

## Structure:
- keyboards/ - Keyboard product images
- mice/ - Mouse product images  
- headphones/ - Headphone product images
- monitors/ - Monitor product images
- mousepads/ - Mousepad product images

## Image Naming Convention:
- [product-id]-1.jpg - Main product image
- [product-id]-2.jpg - Secondary product image

## Note:
In a real implementation, these would be actual product photos.
For this demonstration, you would need to add actual image files or update the database.js file to use placeholder image services like:
- https://via.placeholder.com/300x300/007bff/ffffff?text=Product+Image
- https://picsum.photos/300/300
